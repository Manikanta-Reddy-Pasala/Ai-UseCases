## Health Scoring

```
Health Score: 100 (start)

CPU > 90%  вҶ’ -30 (CRITICAL)    CPU > 70% вҶ’ -15 (WARNING)
MEM > 90%  вҶ’ -30 (CRITICAL)    MEM > 75% вҶ’ -10 (WARNING)
DISK > 90% вҶ’ -25 (CRITICAL)    DISK > 75% вҶ’ -10 (WARNING)
LOAD > 2x CPUs вҶ’ -15 (WARNING)

Score >= 80 вҶ’ "healthy"
Score >= 50 вҶ’ "degraded"
Score <  50 вҶ’ "critical"
```
